11. Google Services Integration

Firebase Firestore

Document path: users/{uid}/worlds/default

Structure: { blocks: Block[], updatedAt: string, blockCount: number }

Security rules enforce user-scoped access. Block count capped at 10,000 per write. Debounced 1500ms.

Firebase Anonymous Auth

Called on app mount via signInAnonymously(auth). No email/password required — users start building immediately. UID is stable across sessions.

App runs in Demo Mode when Firebase credentials are not configured.
